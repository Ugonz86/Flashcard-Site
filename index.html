JavaScript

An object-oriented computer programming language commonly used to create interactive effects within web browsers.

Operator

The definition of an operator is a symbol that is used to perform an operation. Most often these operations are arithmetic (addition, subtraction, etc), but not always.

Variables

Variable means anything that can vary. JavaScript includes variables which hold the data value and it can be changed anytime.

Functions

A function is a subprogram designed to perform a particular task. Functions are executed when they are called.

Methods

JavaScript methods are actions that can be performed on objects. A JavaScript method is a property containing a function definition.
